Простой калькулятор


Возможности

- Считает примеры сложения, вычитания, деления и умножения
- Поддерживает скобки
- Не ломается, если ввести буквы или знаки


Как запустить

1. Убедитесь что установлен Python 3 
2. Скачать файл Calculator.py
3. Запустить в терминале


Пример работы 

Введите ваш пример
>> 30*20+50/2+(10-5)
= 630.0
